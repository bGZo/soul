  - 药品类
    collapsed:: true
    - 感冒药（缓解卡他症状）
      collapsed:: true
      - 泰诺，新康泰克，三九感冒灵和连花清瘟颗粒的话看你接不接受解表的说法，体感效果也是有的
    - 解热镇痛/非甾体抗炎药
      collapsed:: true
      - 洛索洛芬钠（乐松），对乙酰氨基酚，依托考昔，布洛芬（缓释片起效慢，急性发作用平片），注意不要空腹服用
    - 止泻药
      collapsed:: true
      - 蒙脱石散，易蒙停（如果确定是肠道感染食物中毒，不要用止泻药）
    - 肠道感染
      collapsed:: true
      - 诺氟沙星
    - 外用止痛贴剂
      collapsed:: true
      - 氟比洛芬贴剂（得百安）
    - 抗过敏药
      collapsed:: true
      - 西替利嗪，氯雷他定（开瑞坦）
    - 晕车药
      collapsed:: true
      - 苯海拉明
